# Acme Co - Client Portal

A modern web application for Acme Co's client management and project tracking needs.

## Project Overview

This portal provides clients with real-time visibility into their project progress, including:
- Project status updates
- Development milestones
- Resource allocation
- Timeline tracking

## Tech Stack

- Frontend: React 18
- Backend: Node.js/Express
- Database: PostgreSQL
- Authentication: JWT
- Hosting: AWS

## Development Workflow

1. Create feature branch from `main`
2. Implement changes with unit tests
3. Submit PR for code review
4. Deploy to staging for QA
5. Merge to main for production deployment

## Getting Started

```bash
npm install
npm run dev
```
